### Planning:

**a. Materials:**
   - 10” x 4” x 15" block of wood of choice (walnut)
   - Rubber bumper strip
**b. Tools/Equipment**
   - Band saw
   - Scroll Saw
   - Sander
   - Sandpaper
   - Router

### Fabrication
   1. Create a stencil of the side profile
   2. Trace the stencil on the block of wood
   3. Cut out the profile
       - Band Saw
       - Scroll Saw 
       - Router
   4. Finish
        - Sanding
        - Varnish
